开发：打开谷歌浏览器扩展程序，直接把src的内容作为“已解压的扩展程序”加载。

使用：<br>
方法一：打开谷歌浏览器扩展程序，直接把src的内容作为“已解压的扩展程序”加载。<br>
方法二：在谷歌应用程序中直接搜“聚光灯”后安装，也可以直接点击打开这个[链接](https://chrome.google.com/webstore/detail/emfhbanibkdnaleekjbejkocpeapahjm)。
